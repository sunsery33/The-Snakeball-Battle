# 球蛇大作战 / The SnakeBall Battle

一个纯前端 Canvas 原型，把“球球大作战”的大小吞噬和“贪吃蛇大作战”的身体碰撞切断合在一起。

## 运行

直接用浏览器打开 `index.html`。

如果要测试联机大厅和房间服务器：

```bash
cd server
npm install
npm run dev
```

然后打开：

```text
http://localhost:3000
```

联机骨架当前支持创建房间、房间码加入、快速匹配，并固定补满 11 条蛇；真正的多人世界同步是下一步工作。

## 操作

- 鼠标移动控制方向，也支持 `WASD` / 方向键。
- 按住 `Space` 合体：身体球会依次沿着球蛇当前运动轨迹向头部加速前进并被吞噬，最终变成一个大球；这会消耗顶部力量条，耗尽后会强制释放。
- 点按 `Space` 也会触发一小段最低能量消耗，不能靠连点白嫖身体不可吃状态。
- 玩家和人机共用同一套合体能量规则；释放动画里身体不可吃的窗口也会继续消耗能量，耗尽后必须恢复到一半以上才能再次合体。
- 松开 `Space` 或力量耗尽后，动画倒放，身体球从头部沿同一条轨迹吐出并重新排成长条，力量条会恢复。
- 吃到绿球后按住 `Space` 会慢慢消化储备质量；这段时间合体速度更慢。
- `Q` 蓄满后分出一个球，自己变长，总质量不变。
- `E` 蓄满后合并一个球，自己变短，总质量不变。
- 按 `F` 从蛇头吐出一颗食物；食物撞到绿色针状大食物会把它往飞行方向推走。
- 按住鼠标左键或 `V` 加速：会从尾部释放少量食物，自己的质量会慢慢变小。
- `P` 暂停，`R` 重开。

## 规则

- 吃地图上的散落食物会增加质量和长度。
- 地图会始终维持固定数量的自然食物；死亡掉落和加速喷出的食物不会挤占自然刷新。
- 普通状态下，每条蛇自己的所有球大小一致，不会出现头大尾小；只有按住 `Space` 合体时会临时变成一个大球。
- 合体时身体质量进入头部，但原身体路径会保留成一条所有人可见的细线；这条线不能被吃，只用来预判释放后的形态。
- 大头撞到更小的敌方身体球，会从撞击点切断对方，尾段变成食物。
- 头部撞到明显更大的头会被吞掉；撞到吃不下的身体会死亡。
- 地图上的绿色针状大食物数量固定，被吃掉后会随机刷新一个。
- 蛇头或身体碰到绿色针状大食物都会吃掉它，迅速变长，单个球会变小，同时总质量增加；按住 `Space` 可以慢慢消化它带来的额外质量。
